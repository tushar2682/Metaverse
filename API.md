# Metaverse API Documentation

## Authentication
- POST `/api/v1/signup`
- POST `/api/v1/login`

## User
- PUT `/api/v1/user/metadata`
- GET `/api/v1/user/metadata/bulk`

## Spaces
- POST `/api/v1/space`
- DELETE `/api/v1/space/:id`
- GET `/api/v1/space/all`

## Admin
- POST `/api/v1/admin/element`
- PUT `/api/v1/admin/element/:id`
- POST `/api/v1/admin/map`
