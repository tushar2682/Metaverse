# Developer Guide

Welcome to the Metaverse 2D development guide.

## Local Development
1. Ensure PostgreSQL is running.
2. Initialize the database using Prisma.
3. Run `npm run dev` to start the server.

## Running Tests
Run all integration tests with:
```bash
npm test
```

## Project Structure
- `/src`: Controllers, routes, and business logic.
- `/prisma`: Database schema and migrations.
- `/Tests`: API integration tests.
